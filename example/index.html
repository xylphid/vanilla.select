Vanilla select

Vanilla select is a simple overlay for select and multiple select in Vanilla-JS.

Use vanilla select

Add beautify class to your select and you shall see some magic.


    <select multiple="multiple" class="beautify" style="width:350px;">
        <option selected="selected">Apple</option>
        <option>Orange</option>
        <option selected="selected">Banana</option>
        <option>Lemon</option>
    </select>
    <select multiple="multiple" class="beautify">
        <option selected="selected">Red</option>
        <option>Yellow</option>
        <option selected="selected">Purple</option>
        <option>Crimson</option>
    </select>
    <select class="beautify">
        <option>Jedi</option>
        <option>Sith</option>
        <option>Clone</option>
        <option selected="selected">Bounty Hunter</option>
    </select>

Options


    vanilla.select.defaults = {
        placeholder: {
            maxLength: 2,
            empty: 'No selection',
            overflow: '{0} items selected'
        }
    }
